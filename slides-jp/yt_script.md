## 本編

### HDDの基本構造

まず、現在のハードディスクがどうやってデータを保存しているか簡単に説明しましょう。

ハードディスクの中には、高速で回転する円盤があります。
その表面には、薄い磁石の膜が塗られています。

この磁石の向き、N極が上向きか下向きか、がデジタルデータの0と1に対応しています。

現在のハードディスクでは垂直磁気記録という方式が使われています。
磁石を縦向きに並べて記録する方法です。

### 書き込みの仕組み

データの書き込みには、ヘッドと呼ばれる小さな電磁石を使います。

ヘッドに電流を流すと磁場が発生して、ディスクの磁石の向きを変えることができます。
これがデータの書き込みです。

ここで大事なのは、記録に使う磁石の性質です。

釘を磁石にくっつけると、釘も磁石になりますよね。
でも、釘の磁力は弱くて、すぐに消えてしまいます。

ハードディスクの記録層には、「頑固な」磁石の材料が使われています。
磁化させるためには大きな磁場が必要な代わりに、一度磁化されると、簡単には向きが変わらない性質を持っているんです。

```latex
% テロップで補足
この「頑固な」性質を「硬磁性」と呼ぶ
```


だから電源を切っても、データが消えずに残るんですね。

### 読み出しの仕組み

読み出しには、磁場で電気抵抗が変わる特殊なセンサーを使います。

ヘッドがディスクの上を移動すると、磁石の向きに応じてセンサーの抵抗値が変化します。
この変化を電気信号として取り出して、0と1のデータを読み取っているんです。

このセンサーの原理は「巨大磁気抵抗効果」と呼び、
2007年にノーベル物理学賞を受賞した発見なんです。

```latex
% TMRはテロップで補足
現代のHDDでは、GMRをさらに発展させた「TMR」というセンサーが使われています。
TMRは「トンネル磁気抵抗効果」の略です。
```

### HDDの限界とBPMのアイデア

ハードディスクは、薄い磁石の膜に情報を記録しています。
でも、記録密度を上げていくと、物理的な限界にぶつかります。
記録密度が、膜を構成する粒の大きさやばらつきに制約されてしまうんですね。

その限界を突破するアイデア。
それが、磁石の膜ではなく、1つ1つ分離したナノサイズの磁石で記録する方法。

これが「ビットパターンドメディア」、略してBPMです。

### BPMをどうやって作るのか

1つ1つの磁石が1ビット。
アイデアとしてはシンプルです。

でも、20ナノメートルの磁石を何十億個も規則正しく並べる。
聞いただけで気が遠くなりますよね。
どうやればいいんでしょう？

### フォトリソグラフィ — 「光で形を作る」技術

基本になるのが、フォトリソグラフィという技術です。

半導体チップの製造にも使われている、光でパターンを作る方法です。

板の上に、光を当てると溶けやすくなる特殊な材料を塗ります。
その上に、一部だけ穴が開いたマスクを被せて光を当てます。

光が当たった部分だけが溶けやすくなるので、液に浸して洗い流します。

すると、マスクの穴のパターンがそのまま転写されるんです。

### フォトリソグラフィの限界

でも、フォトリソグラフィには限界があります。

光の波長より細かいパターンはぼやけてしまいます。

かつて半導体製造で主流だったUVフォトリソグラフィの波長は、約200ナノメートルです。
BPMに必要な20ナノメートル以下のパターンには、この方法では回折限界から難しいんです。

```latex
% テロップで補足
現在の最先端の半導体製造ではEUVという波長13.5nmの短い光が使われています
```

```latex
% テロップで補足
回折限界とは、波長の1/2以下の細かいパターンはぼやけてしまうという限界のこと。
```

```latex
% テロップで補足
EUVは半導体では使われているがHDDとはコスト・形状の要求がまったく違うのでそのまま使うのは難しい
```

### 電子ビームリソグラフィ — 「原版」を描く

そこで使われるのが電子ビームリソグラフィです。

光の代わりに電子のビームを使うんです。
高電圧で加速した電子の波長は光よりはるかに短いので、ナノスケールのパターンが描けます。

ただし、この方法はパターンを1つずつ描いていく方式です。
精密な分、時間がかかる。

HDDのディスク1枚ずつ電子ビームで作るのは、手間とコストがかかりすぎてしまうんですね。

### ナノインプリントリソグラフィ — 「ハンコ」で大量複製

そこで登場するのが、ナノインプリントリソグラフィという技術です。

考え方はハンコと同じです。

ハンコって、一度原本を作れば、押すだけで同じ印を何個でも作れますよね。

まず、ディスクの表面に、熱で柔らかくなる樹脂を塗ります。
これを加熱して柔らかくします。
そこに、電子ビームで作った精密な「原版」をぎゅっと押し付けます。

冷やして固めてから原版を外すと、電子ビームで作ったパターンを再現できる、というアイディアです。

ハンコを押すように、1枚のテンプレートを作れば、そこから同じパターンを大量のディスクに複製できるわけです。

### NILの凸凹から磁石のドットへ — BPM製造プロセス

でも、これだけだと樹脂の凸凹パターンしか作れません。
どうやって磁石のドットにするんでしょう？

まず、基板の上に磁石の膜を作ります。
その上にドットパターンをハンコのように押します。

こうやってできた柱状に盛り上がってる部分がマスクになるんです。

イオンビームで表面を削ると、マスクのない部分の磁石の膜が先に削れます。
結果として柱状のマスクの下だけ、磁石の島が残るんです。

これがBPMのビット島です。

### BPMの優位点 — なぜ連続膜ではダメなのか

では、わざわざこんな手間をかけてBPMを作るメリットは何でしょう？

従来のHDDの記録層は、小さな結晶粒がランダムに集まった磁性薄膜です。
1つのビットの情報を記録するのに、何十個もの結晶粒を使っています。

記録密度を上げるには、この1ビットあたりの面積を小さくする必要があります。

でも、ここに三すくみの関係があるんです。

記録密度を上げたい。
データを熱で消えないようにしたい。
ヘッドの磁場で書き込めるようにしたい。

この3つを同時に満たすのが、とても難しいんです。

密度を上げるために粒を小さくすると、熱で磁化が不安定になります。
安定させるには強い磁石が必要ですが、強すぎるとヘッドで書けなくなる。
安定かつ書き込めるようにするには粒が大きくなるので、今度は記録密度が上がらない。

お互いがお互いに睨み合って動きを取れなくしているような関係ですね。

BPMは、この三すくみを根本から変えるアプローチです。

従来の方式では、1つのビットに何十個もの粒を使うので、粒を小さくするとノイズが増えます。

BPMでは1つの島が丸ごと1ビットです。
島のサイズや位置は精密に制御されているので、粒のランダムさに由来するノイズがありません。

しかも島全体が1つの大きな磁石として振る舞うので、小さくても熱的に安定で、記録密度を上げやすいんです。

### BPMの課題 — 実用化への壁

じゃあBPMは万能かというと、まだまだ課題があります。

#### 課題1: より「頑固な」磁石が必要

記録密度をさらに上げると、島はもっと小さくなります。

小さな島でもデータを安定に保つには、より頑固な磁石の材料が必要です。

でも頑固すぎると、今度はヘッドの磁場では書き込めなくなります。

#### 課題2: HAMRとの組み合わせ

この壁を越えるために、BPMと他の技術を組み合わせる研究が進んでいます。

例えば熱アシスト磁気記録、英語の頭文字をとってHAMR(ハンマー)と呼ばれています。
レーザーで島を一瞬だけ加熱します。

温度が上がると、磁石の頑固さが一時的に弱まります。
その隙にヘッドで書き込むんです。
冷めれば元の頑固な磁石に戻るので、データは安定です。

BPMとHAMRを組み合わせたらどうなるか。
理論的には、1平方インチあたり10テラビット以上の記録密度も視野に入るとされているんです。
これは現在のHDDの10倍以上の密度です。

#### 課題3: 製造コストと精度

もう1つの大きな壁は製造です。

何十億個もの島を、位置もサイズも均一に作らないといけません。
ほんの数ナノメートルのズレがエラーにつながります。

この製造精度を、コストを抑えながら実現するのが、現時点での最大の課題です。

### BPMの現在地と未来

BPMは現在も研究段階にあります。
先にお話ししたHAMRは、2024年に実用化が始まりました。

AIの急速な普及で、世界のデータ量は爆発的に増え続けています。
これだけのデータを保存するには、記録密度の限界を押し上げ、より少ないエネルギーで情報を記録する技術が不可欠です。

BPMはこれらの課題を解決するための有力な選択肢の1つとして、研究が続けられているんです。
