# 動画コンセプト

このシリーズ全体に適用される制作方針です。

## 基本フォーマット

- **尺**: 10-20分程度
- **構成**: 1動画1テーマ
- **スタイル**: 短く、濃く、わかりやすく
- **ターゲット**: 理工系の学部生、大学院生、研究者、社会人
- **コンセプト**: 学部1-2回生程度の前提知識で物理や技術をわかりやすく平易に伝える

## ディレクトリ構造

```
. 
├── slides-jp/  # 動画制作に関する指示書
│   ├── script-outline.md # 動画の脚本のアウトライン
│   └── yt_script.md # 動画の脚本
├── descriptions/  # 動画の概要欄
│   ├── youtube_script_guidelines.md  # 動画の概要欄 (日本語)
│   └── ... # 以下同様に他言語の概要欄
├── captions/  # 字幕
│   ├── mfm-and-bpm-jp.srt  # 日本語字幕
│   └── ... # 以下同様に他言語の字幕
└── docs/ # 動画制作に関する資料 (バージョニング対象外)
    └── xxx.pdf # 論文や技術資料など

```
